# 🚀 AI Placement Preparation Platform

## 🔥 Overview

A full-stack AI-powered platform that helps students prepare for placements with personalized roadmaps, resume analysis, and interview simulation.

---

## ✨ Features

* 📊 AI Resume Analyzer (ATS + scoring)
* 🧠 Smart Roadmap Generator
* 💻 Monaco Code Editor (multi-language)
* 🎯 Interview Simulator with AI feedback
* 📈 Progress Tracking Dashboard

---

## 🛠 Tech Stack

* React
* Node.js
* Express
* MongoDB
* Gemini API
* Monaco Editor

---

## 🚀 Key Highlights

* LeetCode-style coding interface
* AI-based resume scoring system
* Real-time interview simulation
* Multi-language code support
